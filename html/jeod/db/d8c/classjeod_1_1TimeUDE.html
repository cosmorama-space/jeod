Adds a UDE type to the initialization tree when it is appropriate to do so.

Assumptions and Limitations
  • The time type from which the UDE updates must be in the tree above the UDE.
  • If the time type in which the epoch is defined is another UDE, it also must be in the tree above this UDE
  • This function is only called when the UDE is NOT being used to initialize the simulation.

Each time type is initialized from its parent in the initialization tree, except one.

In order to have an absolute reference time, one of the time types must be defined ahead of time. This is called the initializer time. This function initializes the initializer time.
