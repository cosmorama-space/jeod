◆ initialize_tai_to_ut1()

void jeod::TimeConverter_TAI_UT1::initialize_tai_to_ut1 ( void  )
private

The conversion from Atomic Time (TAI) to Universal Time (UT1) involves the addition of value that is a continuous function of TAI.

That value is tabulated at regular points of TAI. This function initializes that table and sets the preliminary values.

Assumptions and Limitations
  • The table does not go into the future.
