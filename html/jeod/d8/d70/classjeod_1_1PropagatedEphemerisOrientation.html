Detailed Description

A PropagatedEphemerisOrientation is an EphemerisOrientation whose state is coupled with the rotational state of a DynBody reference frame.

This class is intended for use by the PropagatedPlanet class. Use outside of the PropagatedPlanet is not sanctioned.

The class acts analogously to the class PropagatedEphemerisPlanet, but for rotation rather than translation. See PropagatedEphemerisPlanet for a description of the behavior of the class.
