99 /**
100  * Represent both an atomic chunk of mass and an interconnected set of masses.
101  * Multiple MassBody objects can be attached to one another in a tree structure.
102  * This connected sets of bodies has a composite set of mass properties.
103  * Sans these connections, a body will have some core mass proprties.
104  *
105  * The MassBody class is one of the key classes in JEOD 2.0. In addition to
106  * representing masses, it is also the parent of the DynBody class, which is
107  * used in JEOD 2.0 to represent vehicles.
108  *
109  * \par Assumptions and Limitations
110  * - Rigid Bodies
111  * - Tree attachment Structure
112  */
