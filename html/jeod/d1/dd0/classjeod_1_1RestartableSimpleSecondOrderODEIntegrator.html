Detailed Description

template<unsigned int size>
class jeod::RestartableSimpleSecondOrderODEIntegrator< size >

A RestartableSimpleSecondOrderODEIntegrator is-a RestartableSecondOrderODEIntegrator that manages the integrator for a simple second order ODE problem, one in which the time derivative of the generalized position is the generalized velocity.

Template Parameters
sizeSize of the position vector (and also of velocity).
